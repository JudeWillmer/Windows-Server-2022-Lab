## Completed Labs

### Lab 1 - Active Directory User Management Basics

**Objective**

Practice common Active Directory administrative tasks performed by IT Support professionals.

**Tasks Completed**

- Created an Active Directory user account
- Reset a user password
- Disabled a user account
- Unlocked a locked user account
- Moved a user account to an Organizational Unit (OU)
- Searched for users using Find Users, Contacts and Groups
- Verified deleted objects using the Active Directory Recycle Bin

**Skills Demonstrated**

- Active Directory user administration
- Password management
- Account lockout management
- Account lifecycle management
- Organizational Unit (OU) management
- Active Directory object recovery

# Lab 2 - Group Policy Management

**Objective**

Learn how to configure and manage Group Policy Objects (GPOs) to enforce security settings and standardize Windows environments across a Windows Server domain.

**Tasks Completed**

- Reviewed Default Domain Policy security settings
- Configured a Group Policy to enforce a default lock screen and logon image
- Configured the Account Lockout Duration security policy
- Navigated Administrative Templates and Security Settings
- Managed Group Policy using the Group Policy Management Console (GPMC)

**Skills Demonstrated**

- Group Policy Management
- Windows Security Policy Configuration
- Account Lockout Management
- Administrative Templates
- Windows Server Administration
- Enterprise Policy Management

# Lab 3 - Shared Folders & NTFS Permissions

## Objective

Learn how to create SMB network shares, configure NTFS and Share permissions, and provide secure folder access for Active Directory users in a Windows Server environment.

---

## Tasks Completed

* Created an SMB network share using Server Manager
* Verified the shared folder and network path
* Configured NTFS (Security) permissions for an Active Directory user
* Configured SMB Share permissions with Read/Write access
* Applied user access permissions for a shared folder

---

## Skills Demonstrated

* Windows Server File and Storage Services
* SMB Share Management
* NTFS File System Permissions
* Share Permission Configuration
* Active Directory User Access Management
* Windows Server Administration

# Lab 5 - Installing RSAT Tools on Windows 10

## Objective

Install Remote Server Administration Tools (RSAT) on Windows 10 to enable remote management of Windows Server roles such as Active Directory.

## Tasks Completed

- Installed Remote Server Administration Tools (RSAT)
- Opened Active Directory Users and Computers (ADUC)
- Verified the RSAT management console launched successfully
- Prepared a Windows 10 workstation for remote server administration

## Skills Demonstrated

- Windows 10 Administration
- RSAT Installation
- Microsoft Management Console (MMC)
- Active Directory Administration Tools
- Remote Administration Preparation

## Notes

> RSAT installed successfully and Active Directory Users and Computers opened correctly. Remote management functionality was not yet available because the Windows 10 workstation had not yet been fully configured to communicate with the Windows Server domain. This will be completed in a later lab.

# Lab 6 - Joining a Windows 10 PC to an Active Directory Domain

## Objective

Join a Windows 10 workstation to an Active Directory domain and verify the computer object is successfully created in Active Directory.

## Tasks Completed

- Joined a Windows 10 Pro workstation to the Active Directory domain
- Authenticated the domain join using domain administrator credentials
- Restarted the workstation to complete the domain join process
- Verified the computer object appeared in the **Computers** container within Active Directory Users and Computers (ADUC)

## Skills Demonstrated

- Active Directory Domain Administration
- Domain Join Configuration
- Windows 10 Administration
- Computer Object Management
- Windows Server 2022 Administration
